# Dumbcast

A minimalist podcast app for Android feature phones, built with vibes and targeted at the **Sonim XP5800** running Android 8.1.

## Features

- **D-pad optimized UI** - Navigate with physical keys
- **3x3 grid podcast view** - Number keys (1-9) for quick access
- **Episode management** - NEW, BACKLOG, AVAILABLE states with automatic decay
- **Download & offline playback** - Save episodes for offline listening
- **Reverse episode order** - For episodic podcasts (story podcasts)
- **Smart episode state** - Auto-remove from backlog on completion
- **No fluff** - Just podcasts, no unnecessary features

## Target Device

**Sonim XP5800**
- Android 8.1 Oreo
- Physical keypad & D-pad
- Rugged feature phone form factor
- Perfect for minimal, focused podcast listening

## Development

This app is vibe coded - built iteratively with AI assistance focusing on practical functionality over perfect architecture. It works, it's useful, and it respects your attention.

Built for people who want to listen to podcasts without smartphone distractions.
